    Meet RexZ, a powerfull and superfast security bot of the discord.We wanted to give you guys the best commands possible (and there are a lot more planned!) but with only one goal: We wanted both newbies and advanced users to be able to use RexZ's commands. Each command has the basic use and the advanced one to provide the most possible customizability you guys want.

    There are 5 categories regarding RexZ's commands.

    • General:-These commands do not necessarily cover RexZ's functionality. They simply provide more help, avatar and etc.
    • Moderation:- This covers all of RexZ's awesome moderation commands like Kick, ban, setprefix, timeout, ,unban and etc. These commands provide pretty much everything you need in terms of moderation!
    • Security:-This covers pretty much all the security commands that cover all of RexZ's features and systems. Anything that changes RexZ's behaviour in a way is covered here.
    • Trustables:-This covers the functionality like whitelist and unwhitelist etc
    • Automatic Setup:-This covers how to setup RexZ in one second by using -auto
